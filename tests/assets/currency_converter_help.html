Currency Converter API instructions.

Endpoint:

'api/convert?amount=AMOUNT&src-curr=SRC_CURR&dst-curr=DST_CURR&date=DATE'
  • amount: the amount to convert (e.g. 12.35);
  • src-curr: ISO currency code for the source currency to convert (e.g. EUR, USD, GBP);
  • dst-curr: ISO currency code for the destination currency (e.g. EUR, USD, GBP);
  • date: reference_date for the exchange rate, in YYYY-MM-DD format.

Returns

{"amount": AMOUNT, "currency": CUR}

  • amount: the converted amount;
  • currency: ISO currency code for the currency the amount is related to. currency: ISO currency code for the currency the amount is related to.
